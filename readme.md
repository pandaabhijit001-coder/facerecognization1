# Person Finder in Video - Flask App

## Features

- Upload local videos or provide YouTube URLs
- Upload a reference image of a person
- Detect if the person appears in the video
- Shows timestamps and matching face thumbnails
- User registration and login
- Rate limited API endpoints
- Automatic cleanup of old uploaded files
